# Code-Farmar
Syntax Siege is a retro-inspired educational coding game where players control a robot by selecting various code blocks to complete given challenges and puzzles.
 - It is a Idle Game Were You Have to Write Code to Make More Money.
 - It Will Be set in a Island Were you have to Gather Resources
